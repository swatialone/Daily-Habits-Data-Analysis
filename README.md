 # 📊 Daily Habits & Screen Time Data Analysis

Hi! Welcome to my Data Analyst project. In this project, I analyzed a week's worth of daily habit data, including study hours, screen time, sleep, and physical activity (steps).

## 📂 Dataset Overview
The project uses a dataset named `habits_data.csv` which tracks:
- **Study Hours** vs **Screen Time Hours**
- **Sleep Hours** & **Daily Steps Count**

## 💡 Key Insights from the Data
After analyzing the data, here are the top findings:
1. 📈 **Highest Screen Time:** Sunday had the maximum screen time of **5.0 hours**, which makes sense as it was a weekend!
2. 📚 **Peak Study Day:** Thursday was the most productive day with **4.5 hours** of focused study.
3. 🏃‍♂️ **Most Active Day:** Friday recorded the highest step count of **9,000 steps**.
4. 😴 **Best Sleep:** Sunday provided the most rest with **9.0 hours** of sleep.
